# Svelte Web Components

A largely straightforward approach for crafting Svelte Web Components

## Introduction

```
npx degit FlokiTV/svelte-webcomponent my-svelte-app
cd my-svelte-app
npm i
```

## Development
to watch svelte and tailwind
```
npm run dev
```

to serve `/dist` folder
```
npm run serve
```

build only svelte
```
npm run build
```

build only tailwind
```
npm run build-css
```
